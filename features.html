Bootstrap Components & Dynamic Features Used

This page provides a detailed breakdown of the Bootstrap 5.3.3 components integrated into the portfolio website, including their usage locations and purposes. It also covers custom dynamic features implemented with JavaScript for enhanced interactivity.

Dynamic Features (JavaScript)

Description: Cycles through a list of focus topics in the hero section.

Implementation: Uses setInterval to update the text content every 3.2 seconds. Topics include 'Blazor WebAssembly on Azure', 'Bootstrap-driven UI systems', etc.

Purpose: Adds dynamic emphasis to key skills without overwhelming the user.

Description: Allows users to copy the email address with a click.

Implementation: Uses navigator.clipboard.writeText in an onclick handler, with a success alert. Applied to email elements in Contact section and Modal.

Purpose: Improves user experience by simplifying contact actions, with cursor:pointer and hover underline for affordance.

All other interactivity (e.g., collapses, modal, navbar toggle) is handled by Bootstrap's built-in JavaScript. Custom JS is kept minimal in assets/js/app.js, focusing on the rotator and copy utility. No external libraries beyond Bootstrap are used for dynamics.
